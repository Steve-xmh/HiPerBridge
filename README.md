## 特性

- 极简：1MB 不到的应用体积，袖珍级别的 UI，极少的内存用量
- 方便：输入入网令牌即可开始联机畅玩
- 扩展：提供了一个相对强大的插件扩展能力，可以[在此查阅开发说明](./PLUGIN.md)

## 开发警告

本源代码使用了闭源的 Sharp Craft Launcher 组件库，故本源代码的 UI 部分是不能编译的，你需要自行改写 UI 方可正确编译通过。

## 开源协议

本源代码使用 AGPL 3.0 开源协议，如需二次开发且分发请注意开源。
